## Run Seatr using CLI TWO

- **Fork** and **Clone** Repository. Only word from your feature branch.

- Open the project directory.
- Install dependencies.

```bash
npm install
```

- To run the production build

```bash
npx expo
```

- To run tests

```bash
npm run test
```
